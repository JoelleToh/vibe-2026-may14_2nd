
# Rooftop Gardens in Singapore Web App

## Overview

This web application showcases the beautiful rooftop gardens of Singapore. It provides a gallery of stunning images, detailed information about each garden, and an interactive map to help users locate them.

## Project Outline

### Design and Styling

*   **Layout:** A clean, modern, and responsive layout that works on both mobile and web.
*   **Colors:** A vibrant color palette inspired by nature, with a primary green, a secondary teal, and an accent lime green.
*   **Typography:** A clear and readable sans-serif font for all text.
*   **Visuals:** High-quality images of the gardens are used throughout the application.
*   **Icons:** The app uses modern, interactive iconography to enhance user understanding and navigation.
*   **Effects:** Subtle animations and transitions are used to create a more engaging user experience.
*   **Texture:** A subtle noise texture is applied to the main background to add a premium, tactile feel.

### Features

*   **Hero Section:** A full-width hero section with a captivating image and a welcoming message.
*   **Gallery:** A responsive image gallery showcasing the beauty of the gardens.
*   **Garden Information:** A section with cards providing details about each garden, including its name, description, and location.
*   **Interactive Map:** An embedded Google Map with markers indicating the location of each garden.
*   **Navigation:** A sticky navigation bar that allows users to easily jump to different sections of the page.
*   **Footer:** A simple footer with copyright information.
